## ✨ Features

<table>
  <tr>
    <td align="center">🔄</td>
    <td><b>Rotating Orbital Rings</b> - Three dashed rings rotate at different speeds and directions</td>
  </tr>
  <tr>
    <td align="center">💫</td>
    <td><b>Floating Service Nodes</b> - Each AWS service gently floats up and down</td>
  </tr>
  <tr>
    <td align="center">🔗</td>
    <td><b>Dynamic SVG Connectors</b> - Energy lines flow from core to each service</td>
  </tr>
  <tr>
    <td align="center">🎯</td>
    <td><b>Interactive Click Events</b> - Click on any node for cool animations and alerts</td>
  </tr>
  <tr>
    <td align="center">🌈</td>
    <td><b>Color-Coded Services</b> - Each service has its unique color scheme</td>
  </tr>
  <tr>
    <td align="center">✨</td>
    <td><b>Hover Effects</b> - Nodes ripple and grow when hovered</td>
  </tr>
  <tr>
    <td align="center">📱</td>
    <td><b>Fully Responsive</b> - Looks great on mobile, tablet, and desktop</td>
  </tr>

## 🚀 Quick Start

### Option 1: View Live Demo (Easiest)
Just click here: **[https://junaidtyagi9555.github.io/AWS-Orbital-Hero-Page/](https://junaidtyagi9555.github.io/AWS-Orbital-Hero-Page/)**

### Option 2: Run Locally
```bash
# Clone the repository
git clone https://github.com/junaidtyagi9555/AWS-Orbital-Hero-Page.git

# Navigate to project folder
cd AWS-Orbital-Hero-Page

# Open in browser (choose one)
open index.html        # Mac
start index.html       # Windows
xdg-open index.html    # Linux
